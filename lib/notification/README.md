# Notification

## Development

**yarn**

```bash
yarn install
```

```bash
yarn dev
```

**npm**

```bash
npm install
```

```bash
npm run dev
```

## Production

**yarn**

```bash
yarn install
```

```bash
yarn build
```

```bash
yarn prod
```

**npm**

```bash
npm install
```

```bash
npm run build
```

```bash
npm run prod
```

## Release

The one who has permission will push the final JS file from `dist/` to the cloud.
